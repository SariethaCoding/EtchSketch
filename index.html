Help

Etch-A-Sketch Help

Drawing
Click anywhere on the grid to start drawing, and click again to stop
The "Drawing" symbol on the side panel will indicate whether drawing is on / off

Grid Size
Select the size of the grid by adjusting the slider
Changing the size of the grid will reset the board

Colours
Select any colour you wish to draw with by clicking on the colour selector and selecting a colour
"Rainbow Mode" will select random colours for every block while drawing
"Shading Mode" will darken the colours as you move over them (drawing must be active)
